# 04. 画面要件

## 4.1 画面一覧

| ID | 画面 | 概要 |
|---|---|---|
| SC-01 | ログイン画面 | ID/パスワード入力 |
| SC-02 | メイン画面 | 3ペイン構成の文書閲覧・編集画面(本システムの中心) |
| SC-03 | 履歴画面 | 文書の履歴一覧・差分表示・復元(メイン画面内のモーダル/サイドパネルでも可) |
| SC-04 | 検索結果画面 | 全文検索の結果一覧(メイン画面内表示でも可) |
| SC-05 | ユーザー管理画面 | 管理者用。ユーザーの追加・無効化・パスワードリセット |
| SC-06 | 設定画面 | 個人設定(パスワード変更等) |
| SC-07 | ごみ箱画面 | 削除文書の一覧・復元・完全削除(メイン画面内表示でも可) |

## 4.2 メイン画面レイアウト(SC-02)

```
+--------------------------------------------------------------+
| ヘッダー: ロゴ | 検索ボックス | 更新確認 | ユーザー名 | ログアウト |
+----------------+---------------------------------------------+
| サイドバー      | 編集ペイン                                   |
| [フォルダ|タグ] | +-----------------------------------------+ |
|                | | 文書タイトル   編集者/更新日時  [履歴][編集]| |
| ▼ folder-a     | +-----------------------------------------+ |
|   ・文書1      | | ツールバー(編集時): 見出し B I リスト 表 …| |
|   ・文書2      | +-----------------------------------------+ |
| ▼ folder-b     | |                                           | |
|   ・文書3      | |   WYSIWYGエディタ / 閲覧ビュー             | |
|                | |                                           | |
| (タグタブ)     | |                                           | |
|  #設計 (12)    | |                                           | |
|  #議事録 (8)   | |                                           | |
+----------------+-+-----------------------------------------+-+
| ステータスバー: 保存状態 | ロック状態(「◯◯さんが編集中」等)   |
+--------------------------------------------------------------+
```

- サイドバーは「フォルダ」タブと「タグ」タブを切り替え表示(または上下2分割)。幅はドラッグで調整可能。
- サイドバーは折りたたみ可能とし、編集領域を広く使えること。

## 4.3 画面要件詳細

### フォルダペイン

- フォルダ階層をツリー表示し、展開/折りたたみができる。
- 右クリック(コンテキストメニュー)で新規文書・新規フォルダ・リネーム・移動・削除を提供する。
- 現在開いている文書をツリー上でハイライトする。

### タグペイン

- 全タグを件数付きで一覧表示する。
- タグをクリックすると該当文書一覧を表示する。複数タグでのAND絞り込みができるとよい(SHOULD)。

### 編集ペイン

- 閲覧モードと編集モードを持つ。閲覧モードではレンダリング済み文書を表示し、「編集」ボタンで編集モードへ移行(この時点でロック取得)。
- Mermaidコードブロックは図としてレンダリングする(閲覧・編集プレビューとも)。
- `file://` 等の外部ファイルリンクには「パスをコピー」ボタンを表示する。
- 編集モードではツールバーとWYSIWYGエディタを表示する。
- 保存状態(保存済み/未保存/自動保存済み)を常時表示する。
- 他ユーザーが編集中の文書は編集ボタンを無効化し、「◯◯さんが編集中」と表示する。

### 履歴表示(SC-03)

- 履歴一覧: 日時・編集者・(あれば)コメントを新しい順に表示。
- 版を選択すると内容プレビューと現在版との差分(追加/削除のハイライト)を表示。
- 「この版に戻す」ボタンで復元(確認ダイアログあり)。

### 更新確認(外部変更の手動取り込み)

- ヘッダーの「更新確認」でライブラリの再スキャンを実行し、外部ツール(生成AI等)による直接変更を即時取り込む(FR-DOC-08)。
- 通常はファイルシステム監視で自動反映されるため、監視が効かない環境向けの補助操作という位置づけ。

## 4.4 操作性・UI全般

- UI言語は日本語。
- キーボードショートカット(FR-EDIT-04)はブラウザ既定動作(Ctrl+S のページ保存等)を抑止して機能させる。
- 削除等の破壊的操作には確認ダイアログを表示する。
- 画面はPC向けレイアウト(最小幅1280px程度を想定)。モバイル対応はスコープ外。
