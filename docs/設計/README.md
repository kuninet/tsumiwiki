## 設計上の主要な決定

| 論点 | 決定 | 詳細 |
|---|---|---|
| リポジトリ構成 | pnpmモノレポ(client / server / shared) | 01章 |
| Git操作 | simple-git(Git CLI利用。サーバーにGit for Windowsを導入) | 06章 |
| フロントマターの扱い | エディタには本文のみ渡し、フロントマターはサーバーで分離管理 | 05章 |
| Markdown変換 | Tiptap + 独自シリアライザ(往復変換テストで保証) | 05章 |
| 状態管理 | TanStack Query(サーバー状態)+ Zustand(UI状態) | 04章 |
| 認証方式 | セッションCookie(httpOnly)+ bcrypt | 01章・03章 |
