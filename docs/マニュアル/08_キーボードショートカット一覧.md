# 08. キーボードショートカット一覧

TsumiWikiで使えるキーボードショートカットの一覧です。

macOSでは `Ctrl` を `Cmd`(⌘)に読み替えてください。日本語入力(IME)の変換中はショートカットが発火しないよう抑制されています。

## 8.1 グローバル(閲覧時)

| キー | 動作 |
|---|---|
| `Ctrl/Cmd + K` | ヘッダー検索ボックスにフォーカス |
| `Ctrl/Cmd + Z` / `Shift + Z` | ブラウザ標準の Undo / Redo(編集モード内でも有効) |

## 8.2 エディタ(編集モード)

| キー | 動作 |
|---|---|
| `Ctrl/Cmd + S` | 保存 |
| `Ctrl/Cmd + B` | 太字 |
| `Ctrl/Cmd + I` | 斜体 |
| `Ctrl/Cmd + U` | 下線 |
| `Ctrl/Cmd + K` | リンク挿入ダイアログを開く |
| `Ctrl/Cmd + Z` / `Shift + Z` | 取り消し / やり直し |
| `Backspace` | 空リスト項目末で押すとリスト解除 → 段落(Obsidian互換) |
| `[[` | ウィキリンク候補ポップアップを表示 |
| `Enter` | ウィキリンク候補ポップアップから候補を確定 |
| `Escape` | ウィキリンク候補ポップアップ / ダイアログを閉じる |

未実装:
| キー | 動作 |
|---|---|
| `Ctrl/Cmd + P` | 文書名クイックオープン(将来対応予定) |

## 8.3 フォルダツリー(行にフォーカスがある状態)

| キー | 動作 |
|---|---|
| `↑` / `↓` | フォーカス移動 |
| `→` | フォルダ展開 |
| `←` | フォルダ折り畳み(またはアウトデント) |
| `Enter` | 文書=開く / フォルダ=展開 |
| `F2` | リネーム |
| `Delete` | ゴミ箱へ移動 |
| `Ctrl/Cmd + クリック` | 選択トグル(複数選択) |
| `Shift + クリック` | 範囲選択 |

## 8.4 検索・候補ポップアップ

| キー | 動作 |
|---|---|
| `↑` / `↓` | 候補移動 |
| `Enter` | 候補を確定・遷移 |
| `Escape` | 閉じる |

対象: ヘッダー検索ドロップダウン、テンプレートピッカー、ウィキリンク候補ポップアップ

## 8.5 履歴パネル

| キー | 動作 |
|---|---|
| `Escape` | パネルを閉じる |

## 8.6 ページ離脱時

| 状況 | 動作 |
|---|---|
| 編集中で dirty(未保存) | `beforeunload` でブラウザ標準の離脱確認を表示 |
| SPA内の別ページへ遷移 | `confirmNavigationIfDirty()` で確認 |
| タブ閉じ・再読み込み | `pagehide` で下書き保存とロック解放を非同期送信(keepalive fetch) |

## 8.7 マウス操作の主な補足

| 操作 | 動作 |
|---|---|
| フォルダ / 文書のドラッグ&ドロップ | フォルダ移動、複数選択なら一括移動 |
| ツリー空白領域へドロップ | ルート直下に移動 |
| フォルダ行 / 空白領域を右クリック | コンテキストメニュー(新規・リネーム・削除・グルーピング) |
| タグチップ(閲覧時) | クリックでタグ絞り込みモード |
| タグチップ(編集時) | クリックでインラインrename、× で削除 |
| ヘッダーロゴクリック | メイン画面(未選択状態)へ |
| Wikilink(閲覧時) | 内部文書へ遷移。未解決なら「見つかりません」トースト |
| 外部URLリンク(閲覧時) | 新規タブで開く |
| ファイルパスリンク(閲覧時) | パスをクリップボードにコピー |
